Conciliación Bancaria POS

Este módulo permite conciliar automáticamente las líneas de extracto bancario con los pagos POS que Odoo agrupa al cerrar cada sesión.

Características principales:

  • Rastreo de cada pago POS individual dentro del pago contable agrupado.
  • Wizard de previsualización con motor de emparejamiento por monto.
  • Detección de matches exactos, parciales, ambiguos y sin match.
  • Documentación persistente de cada ejecución de conciliación.
